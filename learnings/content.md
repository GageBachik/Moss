# Content Learnings

## Format Performance (initial assumptions -- update with real data)
- Carousels tend to get more saves than single images
- Short videos (15-30s) get more shares than static content
- Hook in first 2 seconds is critical for video retention

## Platform Notes
- TikTok: Casual tone, trending sounds help, carousel slides work well
- Instagram: Polished visuals, carousel is king for saves
- X: Punchy, concise, thread format for longer stories
- YouTube Shorts: Similar to TikTok but slightly more polished
- Threads: Text-forward, conversational, image carousels supported

## Posting Best Practices
- Post to TikTok first (largest organic reach for new accounts)
- Adapt content per platform, don't just cross-post identical content
- Always include a hook/problem statement in the first slide/frame
- CTA should be specific: "Follow for launch day" > "Check it out"
